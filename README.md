Чтобы запустить  программу нужно ввести в терминал следующие команды:
# Сначала соберите jar
mvn clean package

# Запустите jar
java -jar target\demo-0.0.1-SNAPSHOT.jar

Потом зайти в браузер и ввести следующий адресс: http://localhost:8081

Чтобы проверить работоспособность функций требуется добавить после адресса одно из следующих значений:
/hello - откроет страницу со следующим сообщением "Hello World! *_* "
/test - откроет страницу со следующим сообщением "Test endpoint works! :3 "
/sum?a=5&b=3 - откроет страницу с сообщением, где считаеться сумма введённых чисел
/multiply?a=4&b=6 - откроет страницу с сообщением, где считаеться произведение введённых чисел
